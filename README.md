## Аналіз Завдання 2. Обчислення визначеного інтеграла методом Монте-Карло

Мета: обчислити інтеграл функції f(x) = x^2 на відрізку [0, 2] методом Монте-Карло, а також перевірити правильність розрахунків шляхом порівняння з еталонним значенням (аналітичним або з SciPy quad).

- Інтеграл, що обчислюємо: ∫[0,2] x^2 dx = (b^3 − a^3)/3 = (2^3 − 0)/3 = 8/3 ≈ 2.6666666667.
- Алгоритм Монте‑Карло: беремо U ~ Uniform(a, b), тоді оцінка інтеграла I ≈ (b − a) · mean(f(U)).
- Оцінка якості: виводяться стандартна похибка (SE) та 95% довірчий інтервал для оцінки I.

### Як запустити
- Вимоги: Python 3.10+ та NumPy.
- Команда запуску: `python task_2.py`

### Що виводить скрипт
- Кількість вибірок (n).
- Оцінка інтеграла методом Монте‑Карло, стандартна похибка, 95% довірчий інтервал.
- Еталонне значення інтеграла (аналітика для x^2 на [0,2]).
- Абсолютна та відносна похибка оцінки Монте‑Карло відносно еталона.
- Перевірка, чи потрапляє еталонне значення в 95% довірчий інтервал.

Приклад форматованого виводу (ваші числа можуть трохи відрізнятися через випадковість):

```
Monte Carlo method for the integral of f(x)=x^2 on [0, 2]
Number of samples: 200000
Integral estimate (MC): 2.6669xxxx
Standard error (MC): 0.00xxxxxx
95% CI (MC): [2.66xxxxxx, 2.66xxxxxx]

Reference value (analytic): 2.666666666667
Absolute error (MC): 2.3e-04
Relative error (MC): 0.00xxx%
Does the reference value fall within the 95% MC CI? YES
```

### Порівняльний аналіз
- У цьому репозиторії як еталон використовується аналітичне значення інтеграла (8/3) — воно еквівалентне результату SciPy integrate.quad для цієї функції.
- За потреби можна замінити еталон на SciPy quad: функція quad повертає значення та оцінку абсолютної похибки. У нашій реалізації це не обов’язково, оскільки для x^2 аналітичний розв’язок простий і точний.

#### Висновки щодо правильності розрахунків
- Оцінка методом Монте‑Карло наближається до аналітичного значення 8/3. При великій кількості вибірок (наприклад, n = 200 000) абсолютна та відносна похибки є малими.
- Еталонне значення, як правило, потрапляє у 95% довірчий інтервал оцінки Монте‑Карло. Це узгоджується з теорією: ДІ побудований через нормальне наближення зі стандартною похибкою SE.
- Зі збільшенням кількості вибірок n стандартна похибка зменшується приблизно як 1/sqrt(n), що підвищує точність оцінки й звужує ДІ.
